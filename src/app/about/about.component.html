Функционал:

- получение списка постов
 изменения поста сохраняется при смене страницы
список постов обновляется при обновлении страницы

- публикация поста
 вводить можно любые символы
допускается пустое значение
возможность удаления поста
допускается отсутствие постов

- получение списка комментариев для поста
 задание провалено. не справилась с двумерным массивом.
комментарии отображаются как общие для всех постов.

- публикация комментария
 вводить можно любые символы
допускается пустое значение
возможность удаления пкомментария
допускается отсутствие комментариев
